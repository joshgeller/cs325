******************************************

To compile and execute the main program, run the following command:

python3 proj2_main.py [filename]

This will run all 3 algorithms against the problems defined in [filename],
then output the results to [filename]change.txt.

Note: Problems must be formatted the same way they appear in CoinW16.txt, e.g.

[1, 5, 10, 25, 50]
75

******************************************

To compile and execute the experimental analysis script (requires matplotlib 
and numpy libraries), run the following command:

    python3 proj2_analysis.py --plot
